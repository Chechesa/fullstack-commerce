### Ejecución

### `node services.js`
Los servicios (API) funcionan sobre el puerto 4000
### `npm start`
El proyecto funciona sobre el puerto default, la dirección de inicio es:

http://localhost:3000/

Colecciones: productos y usuarios

- Catálogo
- Detalle de producto
- Carrito de compras
- Panel de productos
- Creación de pedidos
